# Metasploitable2

Metasploitable 2  is an intentionally vulnerable Ubuntu Linux virtual machine, designed for testing common 
vulnerabilities (mostly php applications). 
[The virtual machine (VM)](https://docs.rapid7.com/metasploit/metasploitable-2/) is compatible with VMWare, VirtualBox, and 
other common virtualization platforms.

## Using VMWare 

* [Metasploitable](https://information.rapid7.com/metasploitable-download.html)
